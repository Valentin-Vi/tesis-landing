# Resources Directory

Place your videos and images here:

## Videos (in `/videos` subdirectory)
- `hackathon-hero.mp4` - Hero section background video (you at Platanus Build Night)
- `microfauna-demo.mp4` - Microfauna detection project demo
- `scale-ai-process.mp4` - Scale AI process optimization/team leadership footage
- `hospital-platform.mp4` - Hospital management platform demo

## Images (in `/images` subdirectory if needed)
- Platanus hackathon photos
- Project screenshots
- Any other supporting images

## Format Recommendations
- Videos: MP4 format, optimized for web (compressed but good quality)
- Keep file sizes reasonable for web loading (videos under 10-20MB each)
- Videos will play on mute with autoplay

## File Structure
```
public/resources/
├── videos/
│   ├── hackathon-hero.mp4
│   ├── microfauna-demo.mp4
│   ├── scale-ai-process.mp4
│   └── hospital-platform.mp4
└── images/ (optional)
    └── (add as needed)
```

Once you add files here, they'll automatically appear on the landing page.
